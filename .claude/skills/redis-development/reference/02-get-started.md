# Redis Open Source Documentation

## Overview

Redis is an "in-memory data store used by millions of developers as a cache, vector database, document database, streaming engine, and message broker."

The platform features built-in replication and configurable persistence levels, supporting complex data types including strings, hashes, lists, sets, sorted sets, and JSON with atomic operations.

## Installation Options

Multiple deployment paths are available:

- **Linux**: APT, RPM, or Snap package managers
- **macOS**: Homebrew installation
- **Windows**: Docker-based deployment
- **Docker**: Containerized Redis
- **Source**: Compile from source code
- **Redis Stack**: Bundled with Redis Insight

## Primary Use Cases

The documentation highlights five key applications:

1. Data structure storage and retrieval
2. Document database functionality
3. Vector database capabilities
4. AI agents and chatbots
5. Retrieval Augmented Generation (RAG) systems

## Integration Tools & Libraries

Available integrations span:

- Client API libraries across multiple languages
- Redis Data Integration platform
- Python vector library (RedisVL)
- Amazon Bedrock integration
- .NET object-mapping framework
- Spring Data Redis for Java

## Deployment Models

Three primary deployment strategies exist:

1. **Redis Cloud** - Managed service across major cloud providers
2. **Redis Enterprise Software** - Self-managed on-premises or cloud infrastructure
3. **Kubernetes** - Enterprise operator supporting various Kubernetes distributions

Each deployment method includes quickstart guides and detailed configuration documentation.
