# ioredis: Powerful Redis Client for Node.js

## Overview

ioredis is a high-performance Redis client library for Node.js, trusted by enterprises including Alibaba. As stated on the site, it delivers "high-performance, full-featured Redis connectivity for your applications."

## Key Features

**Core Capabilities:**
- Full Redis support including Cluster, Sentinel, Streams, and Pub/Sub
- Lua scripting with abstractions for custom commands
- Binary message support and GEO command functionality
- Advanced pipelining and autopipelining for optimized throughput

**Developer Experience:**
- Works with both Node.js callbacks and native promises
- ES6 type support (Map, Set)
- Clean, approachable API design
- Argument and reply transformation capabilities

**Reliability:**
- Offline queue handling and readiness checks
- TLS/SSL encryption support
- Sophisticated error management strategies
- NAT mapping and TCP/IP or UNIX domain socket support

## Compatibility

- Redis versions: 2.6.12 and later
- Node.js versions: 0.11.16 and later, plus io.js
- Open source under MIT License (free for personal and commercial use)

## ioredis vs node-redis

The site provides a comparison highlighting that ioredis excels with clustering and Sentinel support, while node-redis offers more comprehensive Redis 8 and Stack feature coverage with active maintenance.
