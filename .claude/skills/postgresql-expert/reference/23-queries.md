# Chapter 7. Queries

## Overview

This chapter introduces PostgreSQL's query functionality, which allows users to retrieve data from databases. The documentation explains that after learning how to create tables, populate them with data, and modify that data, the focus shifts to "how to retrieve the data from the database."

## Table of Contents

The chapter covers the following topics:

- **7.1. Overview** - Introduction to query concepts
- **7.2. Table Expressions** - Including FROM, WHERE, GROUP BY, HAVING, GROUPING SETS, CUBE, ROLLUP, and window function processing
- **7.3. Select Lists** - Selection items, column labels, and DISTINCT functionality
- **7.4. Combining Queries** - UNION, INTERSECT, and EXCEPT operations
- **7.5. Sorting Rows** - ORDER BY clause
- **7.6. LIMIT and OFFSET** - Result pagination
- **7.7. VALUES Lists** - Direct value specification
- **7.8. WITH Queries** - Common Table Expressions (CTEs), including recursive queries, materialization, and data-modifying statements

This foundational chapter provides the framework for understanding how to structure and execute queries in PostgreSQL version 18.
