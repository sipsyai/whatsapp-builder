# PostgreSQL 18.1 Documentation: Part I. Tutorial

## Introduction

This tutorial introduces PostgreSQL, relational database concepts, and SQL fundamentals. It assumes general computer knowledge but requires no Unix or programming experience. The content provides "hands-on experience with important aspects of the PostgreSQL system" without attempting comprehensive coverage.

Upon completion, readers should explore Part II for deeper SQL understanding, Part IV for application development guidance, or Part III for installation management topics.
