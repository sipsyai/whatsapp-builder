# Getting Started

Bu bölümde Chatbot Builder'ı kullanmaya başlamak için gereken temel adımları öğreneceksiniz.

## My Chatbots Sayfası

Uygulamaya giriş yaptığınızda **My Chatbots** sayfası karşınıza çıkar. Bu sayfa mevcut chatbot'larınızı listeler ve yeni chatbot oluşturmanıza olanak tanır.

![My Chatbots Listesi](images/01-chatbots-list.png)

### Sayfa Bileşenleri

| Bileşen | Açıklama |
|---------|----------|
| **Import** | JSON formatında chatbot içe aktarma |
| **Create New ChatBot** | Yeni boş chatbot oluşturma |
| **Search** | İsim veya açıklamaya göre arama |
| **Filter** | All ChatBots / Active Only / Archived Only |
| **Statistics** | Total ChatBots, Total Nodes, Total Connections |

### Chatbot Kartı

Her chatbot kartında şu bilgiler ve işlemler bulunur:

- **Status Badge**: Active/Archived durumu
- **Edit Button**: Chatbot bilgilerini düzenleme
- **Download Button**: JSON olarak dışa aktarma
- **Delete Button**: Chatbot'u silme
- **Node/Edge Count**: Akıştaki node ve bağlantı sayısı
- **Created/Updated Date**: Oluşturulma ve güncellenme tarihi
- **Set as Active**: Aktif chatbot olarak ayarlama
- **Load ChatBot**: Builder'da açma

## Builder'a Erişim

Chatbot Builder'a iki şekilde erişebilirsiniz:

1. **Load ChatBot** butonuna tıklayarak mevcut bir chatbot'u yükleme
2. Sol sidebar menüsünden **Builder** sekmesine tıklama

## Sidebar Menüsü

Ana sidebar menüsü uygulamanın tüm bölümlerine erişim sağlar:

| Menü | Açıklama |
|------|----------|
| **My Chatbots** | Chatbot listesi ve yönetimi |
| **Builder** | Görsel akış tasarımcısı |
| **Sessions** | Kullanıcı oturumları |
| **Chat** | Canlı sohbet önizleme |
| **Users** | Kullanıcı yönetimi |
| **WhatsApp Flows** | WhatsApp Flow yönetimi |
| **Data Sources** | Harici veri kaynakları |
| **Settings** | WhatsApp yapılandırması |
