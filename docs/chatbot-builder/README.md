# Chatbot Builder Documentation

WhatsApp Builder platformunun görsel chatbot tasarım aracı olan **Chatbot Builder** için kapsamlı kullanım kılavuzu.

## Overview

Chatbot Builder, WhatsApp chatbot akışlarını sürükle-bırak yöntemiyle tasarlamanıza olanak tanıyan güçlü bir görsel araçtır. React Flow tabanlı canvas üzerinde node'ları bağlayarak karmaşık konuşma akışları oluşturabilirsiniz.

## Quick Start

1. **My Chatbots** sayfasından yeni chatbot oluşturun veya mevcut birini yükleyin
2. **Builder** sayfasında node'ları sidebar'dan canvas'a ekleyin
3. Node'ları birbirine bağlayarak akış oluşturun
4. Her node'u **settings** butonu ile yapılandırın
5. **Validate** ile akışı kontrol edin
6. **Update Flow** ile kaydedin

## Features at a Glance

| Feature | Description |
|---------|-------------|
| Visual Flow Builder | Sürükle-bırak ile akış tasarımı |
| 6 Node Types | Message, Question, Condition, WhatsApp Flow, REST API |
| Real-time Validation | Anlık hata ve uyarı kontrolü |
| Test Mode | Akışı simüle etme |
| AI Build | Yapay zeka ile otomatik akış oluşturma |
| Auto Layout | Otomatik node düzenleme |
