# Whatsapp Builder

This project has been reorganized into a monorepo-like structure.

## Structure

- **client/**: Frontend application built with React, Vite, and React Flow.
- **server/**: Backend application built with NestJS.

## Getting Started

### Client
1. `cd client`
2. `npm install`
3. `npm run dev`

### Server
1. `cd server`
2. `npm install`
3. `npm run start:dev`
